## To initialize project you need:

1. Create a database called 'csvloader'
2.  ```cd api/db/config```
3. Open config.json and conf.js and configure it accordingly
4. ```npm i```
5. ```node server.js```
6. ```node_modules/.bin/sequelize db:migrate```
7. From root directory ```cd public```
8. ```npm i```
9. ```npm run dev```
